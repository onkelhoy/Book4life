Uploading problems

We are terrible sorry that you're having problems with uploading. Hopefully one of these alteritives can help you.

  • How to upload

    Well this is quite an easy problem.
    If you're on a mobile you should see a menu button at the top left corner, press that and a menu should popup with some alteretives choose 'Upload'. If you are on some bigger devices you should see the 'plus button' with an arrow attached to it saying Upload, press that.

    Step by step tutorial

    • Title

      You can search for an existing book and upload a chapter directly to it!

      If the book does not exist, it will be created so be carefull with the name!

    • New Book

      If the book does not exist a popup will occur asking you to put in some info

      author and summery are quite streight foreward, but Tags can be tricky. Tags are only to benifit the search algoritm, let say you wrote a book about mathematics a good tag would be math.
      Tags should be written like this - we,love,tags,at,book4life

    • Chapter

      Don't be fooled with the chapter, a book is built-up by chapters therfore it's important to keep track of which is which.

    • Upload

      When you filled in the info, selected your images it's time to begin the upload!

    • Done

      Now you should hopefully be done!

  • Upload failed

    As we're using the imgur api - we upload the images to www.imgur.com (they are not public), it occurs some problems with traffic. Theres a maximal ammount of requests (about 12,500 uploades per day). But at once you can only upload about 25-50 pages depends on your internet.
    So we strongly recommend to keep the limit to about 30 pages (about one chapter :))
